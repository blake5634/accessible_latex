*** make_accessible.py   A tool to make your latex into accessible html with mathML.

1) install the Pandoc package:  https://pandoc.org/installing.html
2) run   > python3 make_accessible.py --help  for info. 
3) try it on your latex files (make backups first!!)
4) Edit the alternate texts for your \includegraphics figures to make sure they are correct
   and useful. 
5) Compile your new latex and see how it looks
6) Run it through an accessibility checker like the free one at: https://www.accessibilitychecking.com/
4) Thank Claude Code. 

(I have not tested the degree of compliance yet).
